# SomeModsIGuess
This is the SomeModsIGuess add-on for Bedrock Minecraft system.
# Prequisites
When spawning in, use a command block or cheats to run this ```/scoreboard objectives add TameSuccess dummy```. This is required by the Wither Skeleton Captain module.

You also need to enable Beta APIs for some features. If you don't, some features, like the Ore Regeneration module, will not be available.
